## Install

run it manually by first installing [atom-shell](https://github.com/atom/atom-shell/releases), add it to your `PATH` and follow the instructions below:

```sh
$ git clone https://github.com/junmer/fontmin-app.git
$ cd fontmin-app
$ make app
```
